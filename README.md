# Text-based-dungeon-game
Python Dungeon Adventure is a terminal-based game where the player explores different rooms in search of enemies and treasure. The player must defeat enemies, collect rewards, and survive until the final battle.

##  Features

- Hero name selection
- Four rooms to explore
- Random enemy placement
- Attack, Run, and Use Potion options
- Health (HP) system
- Score tracking
- Inventory system
- Treasure rewards
- Final boss battle
- Game Over and Victory screens

##  How to Run

1. Install Python 3.
2. Download or clone this repository.


git clone https://github.com/AyeshaArif006/text-based-dungeon-game.git


3. Navigate to the project folder.

cd text-based-dungeon-game


4. Run the game.

python main.py
